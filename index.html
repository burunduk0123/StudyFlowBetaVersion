Beta Version (Current Stage)

Fully functional StudyFlow application. Features dynamic task creation and a linked calendar grid.

2. Technical Challenges: Debugging

During development, I faced a critical rendering error where the browser displayed raw JavaScript code instead of the UI. This was due to broken HTML tags and scripts being placed outside the body.

Solution: I restructured the file into standard HTML5 sections and ensured all JavaScript logic was encapsulated within <script> tags at the end of the document.
