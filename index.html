Development Process

This section documents the evolution of StudyFlow from an idea to a functional tool.

1. Version Timeline

Alpha Version (March 2026)

Status: Conceptual Prototype.

The product was originally focused on investment tracking. It had a basic table layout but lacked user-friendly interactions.

[Screenshot: Alpha Version Investment Table]

Beta Version (April - May 2026)

Status: Functional Pivot.

Pivot to Study Management. Added a calendar system, progress bars, and tab navigation. Fixed UI issues based on student feedback.

[Demo: Functional StudyFlow Dashboard]

Final Version (Coming Soon)

Planned: Cloud sync, mobile optimization, and AI task prioritization.

2. Feedback Evidence

Alpha Stage (Sponsor Interview)

Video Placeholder: [Sponsor_Interview_Alpha.mp4]

  • Key Question: Would you use an investment tracker?
  • Feedback: "As a student, I need something for homework more than stocks."
  • Result: Pivot to StudyFlow.

Beta Stage (User Testing)

Tested by 2 classmates. Their feedback:

  • User 1: "I need to see my week at a glance." (Added Calendar).
  • User 2: "The 'Complete' button is hard to find." (Updated UI colors).
