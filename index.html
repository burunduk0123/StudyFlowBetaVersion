Project Description

StudyFlow is a productivity platform that helps students organize tasks and improve focus.

Students often struggle with time management and deadlines.

This system helps reduce stress and improve performance.

The idea is unique because it combines planning and progress tracking in one interface.

Full Functionality

Step 1: Add a task with subject, description and date.

Step 2: Tasks are sorted by date.

Step 3: Select and complete tasks.

Step 4: Progress updates automatically.

Step 5: Filter tasks and attach files.

Technical & Functional Requirements

  • HTML, CSS, JavaScript
  • GitHub Pages hosting
  • Task creation and tracking
  • Calendar view
